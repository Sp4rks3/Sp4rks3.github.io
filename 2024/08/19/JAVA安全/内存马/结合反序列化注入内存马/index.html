前言

前一篇写到了Filter内存马，它适用的场景是：允许文件JSP文件上传；并且还有一定缺陷，因为 Web 服务器中的 JSP 编译器会编译生成对应的 java 文件然后进行编译加载并实例化，所以实际上还是会落地的。

配置

因为springboot2停止维护，已经不能通过官方脚手架配置springboot2了，所以使用阿里云的脚手架来配置springboot2；其实之前也准备使用springboot3来写这篇文章，但是确实高版本java限制太多了，例如：反射更加严格、反序列化需要用到的库已经更改了等等一系列问题，解决问题解决的心烦意乱，对我来讲实在是不太友好，所以最终还是选择了springboot2。
